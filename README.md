
MongoLS

=========

A JavaScript implementation of the mongo query api for plain objects and HTML5 localStorage


## Installation

  `npm install mongols`

## Usage

    var numFormatter = require('mongols');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
